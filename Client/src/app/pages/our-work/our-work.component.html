Emotion Recognition Model

Our Emotion Recognition Model is a deep learning solution designed to identify and classify facial expressions into seven distinct emotions. This model has been trained on grayscale images with a resolution of 48x48 pixels.

Dataset

The Emotion Recognition Model is trained on the FER2013 dataset, which is a publicly available dataset for facial expression recognition. It consists of 35,887 grayscale images with a resolution of 48x48 pixels. The dataset is divided into three sets: training, validation, and testing, with approximately 80% of the images used for training, 10% for validation, and 10% for testing.

Model Architecture

model = Sequential() # Convolutional layers
model.add(Conv2D(128, kernel_size=(3,3), activation='relu', input_shape=(48,48,1))) model.add(MaxPooling2D(pool_size=(2,2))) model.add(Dropout(0.4))
model.add(Conv2D(256, kernel_size=(3,3), activation='relu')) model.add(MaxPooling2D(pool_size=(2,2))) model.add(Dropout(0.4))
model.add(Conv2D(512, kernel_size=(3,3), activation='relu')) model.add(MaxPooling2D(pool_size=(2,2))) model.add(Dropout(0.4))
model.add(Conv2D(512, kernel_size=(3,3), activation='relu')) model.add(MaxPooling2D(pool_size=(2,2))) model.add(Dropout(0.4))
model.add(Flatten()) # Fully connected layers
model.add(Dense(512, activation='relu')) model.add(Dropout(0.4)) model.add(Dense(256, activation='relu')) model.add(Dropout(0.3)) # Output layer
model.add(Dense(7, activation='softmax'))
  • Convolutional Layers:
    • Four convolutional layers with 128, 256, and 512 filters, respectively.
    • Each layer uses a 3x3 kernel with ReLU activation.
    • MaxPooling is applied after each convolution layer to reduce the spatial dimensions.
    • Dropout is included after each pooling layer to prevent overfitting.
  • Fully Connected Layers:
    • Two fully connected layers with 512 and 256 units, both using ReLU activation.
    • Dropout is applied after each dense layer for regularization.
  • Output Layer:
    • A final Dense layer with 7 units using softmax activation for multiclass classification.

Model Performance

The model achieves an accuracy of 62% on the test set, indicating its ability to correctly classify facial emotions most of the time.
